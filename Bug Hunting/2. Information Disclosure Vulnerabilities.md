The first vulnerability we learn is **Information Disclosure**.

This type of bug falls under **Cryptographic Failures** in the OWASP Top 10 security threats, where it currently ranks **fourth**.

**OWASP** (Open Web Application Security Project) is an independent, nonprofit organization focused on improving web security. It provides widely used resources based on large-scale testing and real-world data. OWASP publishes rankings and lists to help security professionals understand the most common and dangerous vulnerabilities to watch for.

The term **Information Disclosure** refers to vulnerabilities that allow unauthorized access to sensitive information that should remain private or protected.

Starting with this class of bugs is strategic. They are relatively simple to understand, very common in real applications, and therefore make an excellent starting point for beginners in bug hunting.
